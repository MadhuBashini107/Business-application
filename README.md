# Business-application
# **AIM**
To create a **Warehouse Application** in Salesforce.com.

---

# **Procedure**

## **Types of Applications in SFDC**
In Salesforce.com, there are two types of Applications:

1. **Classic Apps**  
2. **Lightning Apps**

Salesforce allows users to **create, manage, and customize** both app types based on requirements.

---
